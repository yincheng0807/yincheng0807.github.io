Python join()方法

描述

Python join() 方法用于将序列中的元素以指定的字符连接生成一个新的字符串。

语法

join()方法语法：
str.join(sequence)

参数

sequence – 要连接的元素序列。

返回值

返回通过指定字符连接序列中元素后生成的新字符串。
